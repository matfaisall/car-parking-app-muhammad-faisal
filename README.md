## About this Project

FRONTEND TEST : CAR PARKING APP - SIMPLE AND CLEAN

## Getting Started

First, clone the repository:

```bash
git clone https://github.com/matfaisall/car-parking-app-muhammad-faisal.git
```

second, install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
